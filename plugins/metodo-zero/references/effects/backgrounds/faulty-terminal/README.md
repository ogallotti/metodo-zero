# Faulty Terminal

Canvas 2D matrix rain with falling ASCII and katakana characters, CRT scanlines overlay, random glitch blocks, and horizontal screen tears.

## When to use

- Hacker/cyberpunk-themed sections
- Developer tool landing pages
- Sections needing a techy, dystopian atmosphere

## Integration

```html
<div class="ftr-container">
  <canvas class="ftr-canvas"></canvas>
  <div class="ftr-scanlines"></div>
  <div class="ftr-content">
    <!-- Your content here -->
  </div>
</div>
```

## Notes

- Uses Canvas 2D with semi-transparent fade for trail effect
- Scanlines are a pure CSS repeating gradient overlay
- Character set includes ASCII and katakana for authentic Matrix feel
- Random glitch blocks and horizontal tears add CRT malfunction aesthetic
