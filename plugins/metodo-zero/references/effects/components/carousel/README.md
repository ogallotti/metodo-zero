# Carousel

Smooth carousel with drag support, dots, and arrows. Touch-friendly.

## CSS Custom Properties

| Property | Default | Description |
|-|-|
| `--car-radius` | `16px` | Border radius |
| `--car-accent` | `#6366f1` | Active dot color |

## JS Parameters

| Param | Default | Description |
|-|-|
| `autoplay` | `0` | Autoplay interval (0 = off) |
