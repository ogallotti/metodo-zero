# Bento Grid

Animated bento-box grid layout with staggered reveal on scroll.

## Usage

Add `.bg-grid` with `.bg-cell` children. Use `.bg-cell--span-2` and `.bg-cell--span-row` for spanning.

## CSS Custom Properties

| Property | Default | Description |
|-|-|
| `--bg-gap` | `16px` | Grid gap |
| `--bg-radius` | `16px` | Cell border radius |
| `--bg-stagger` | `100ms` | Stagger delay between cells |
