# Ripple Button

Material-design ripple expanding from exact click point. Ripple span is dynamically created and removed after animation completes.

## Usage

Add `.ripple-btn` class. JS handles ripple creation on click. Variants: `--outline`, `--dark`.

## CSS Custom Properties

| Property | Default | Description |
|-|-|
| `--ripple-bg` | `#6366f1` | Button background |
| `--ripple-color` | `rgba(255,255,255,0.35)` | Ripple wave color |
| `--ripple-duration` | `600ms` | Ripple animation speed |
| `--ripple-radius` | `12px` | Border radius |
