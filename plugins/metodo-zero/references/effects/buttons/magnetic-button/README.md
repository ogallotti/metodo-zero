# Magnetic Button

Button magnetically pulls toward cursor when within a configurable trigger area. Text element moves independently for parallax depth.

## Usage

Wrap `.mag-btn` in `.mag-btn-area` (enlarged hit area). Add `.mag-btn__text` span inside button for text parallax.

## CSS Custom Properties

| Property | Default | Description |
|-|-|
| `--mag-bg` | `#6366f1` | Button background |
| `--mag-color` | `#ffffff` | Text color |
| `--mag-radius` | `12px` | Border radius |
| `--mag-padding-x` | `36px` | Horizontal padding |
| `--mag-padding-y` | `14px` | Vertical padding |
