# Marquee

Smooth infinite marquee text scrolling. CSS-only animation with JS for duplication.

## Usage

Add `.mq-container` with `.mq-track` containing `.mq-text` spans. Script duplicates content for seamless loop.

## CSS Custom Properties

| Property | Default | Description |
|-|-|
| `--mq-speed` | `20s` | Scroll duration |
| `--mq-gap` | `48px` | Text spacing |
| `--mq-font-size` | `4rem` | Text size |
| `--mq-color` | `#fafafa` | Filled text color |
| `--mq-stroke-color` | `#3f3f46` | Outline text color |
