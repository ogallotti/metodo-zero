# Lens Zoom

Magnifying lens effect on hover over an image.

## Usage

Add `.lz-container` with an `img` and `.lz-lens` div. Script handles zoom positioning.

## CSS Custom Properties

| Property | Default | Description |
|-|-|
| `--lz-lens-size` | `180px` | Lens diameter |
| `--lz-border-color` | `#ffffff` | Lens border |
| `--lz-border-width` | `3px` | Border thickness |

## JS Parameters

| Param | Default | Description |
|-|-|
| `zoom` | `2.5` | Magnification level |
