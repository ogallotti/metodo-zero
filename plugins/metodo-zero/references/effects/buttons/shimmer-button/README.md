# Shimmer Button

Continuous shimmer/shine sweep across button surface. Pure CSS animation using `::before` pseudo-element with linear gradient.

## Usage

Add `.shimmer-btn` class. Variant: `--lg` for larger size.

## CSS Custom Properties

| Property | Default | Description |
|-|-|
| `--shim-bg` | `#1a1a2e` | Button background |
| `--shim-color` | `#e0e0e0` | Text color |
| `--shim-shine` | `rgba(255,255,255,0.12)` | Shimmer color |
| `--shim-speed` | `2.5s` | Animation speed |
| `--shim-radius` | `12px` | Border radius |
