# Timeline

Animated vertical timeline with entries that reveal on scroll.

## Usage

Add `.tl-timeline` with `.tl-entry` items, each containing a `.tl-dot`. Line fill grows as entries are revealed.

## CSS Custom Properties

| Property | Default | Description |
|-|-|
| `--tl-line-color` | `#27272a` | Timeline line color |
| `--tl-accent` | `#6366f1` | Active dot and line fill color |
| `--tl-dot-size` | `14px` | Dot diameter |
