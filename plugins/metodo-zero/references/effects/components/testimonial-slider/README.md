# Testimonial Slider

Testimonial carousel with avatars, quotes, and smooth crossfade transitions.

## CSS Custom Properties

| Property | Default | Description |
|-|-|
| `--ts-accent` | `#6366f1` | Accent color |
| `--ts-radius` | `16px` | Card border radius |

## JS Parameters

| Param | Default | Description |
|-|-|
| `autoplay` | `5000` | Autoplay interval (0 = off) |
