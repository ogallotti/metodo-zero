# Referencia Criativa Extensiva

Este arquivo contem TODAS as possibilidades de layout, composicao, movimento e interacao da web moderna. Use como catalogo para variar seus designs.

---

## ARQUETIPOS DE COMPOSICAO

### Baseados em Grid
- Grid Simetrico → Colunas iguais, ordem visual clara
- Grid Assimetrico → Colunas de tamanhos diferentes, tensao visual
- Masonry → Alturas variadas como Pinterest, densidade organica
- Bento Box → Celulas de tamanhos variados como caixa de bento japones
- Modular → Modulos que se repetem com variacoes
- Broken Grid → Elementos que intencionalmente quebram a grade
- Overlapping Grid → Elementos que transbordam suas celulas
- Responsive Columns → Colunas que mudam dramaticamente entre breakpoints
- Golden Ratio → Proporcoes auricas, equilibrio natural
- Rule of Thirds → Composicao fotografica aplicada a layout

### Baseados em Divisao
- Split Vertical → Divisao 50/50 lado a lado
- Split Horizontal → Divisao topo/base
- Split Diagonal → Divisao em angulo
- Split Assimetrico → Divisao 70/30 ou 60/40
- Split com Overlap → Divisao onde conteudos se cruzam
- Multi-Split → Mais de 2 divisoes
- Split Animado → Divisao que muda com scroll/interacao
- Split Revelador → Uma parte revela a outra

### Baseados em Camadas
- Layered → Elementos empilhados em Z com profundidade
- Glassmorphism → Camadas translucidas com blur
- Card Stack → Cards empilhados como baralho
- Parallax Layers → Camadas que movem em velocidades diferentes
- Depth Blur → Camadas com desfoque progressivo
- Shadow Depth → Sombras que criam ilusao de profundidade
- Floating Cards → Elementos que parecem flutuar sobre fundo
- Nested Frames → Molduras dentro de molduras

### Baseados em Fluxo
- Scroll Cinematico → Scroll como timeline de filme
- Scroll Horizontal → Movimento lateral ao inves de vertical
- Scroll Snap → Secoes que "travam" na tela
- Infinite Scroll → Conteudo que parece nao ter fim
- Scroll Storytelling → Narrativa que se desenrola com scroll
- Scroll Triggered → Eventos que acontecem em pontos do scroll
- Scroll Hijacking → Scroll controlado com animacoes
- Locomotive Scroll → Scroll suavizado com inertia

### Baseados em Foco
- Hero Dominante → Hero ocupa 100vh, resto e secundario
- Single Focus → Uma unica coisa por secao
- Spotlight → Elemento central com tudo ao redor escurecido
- Progressive Reveal → Foco muda conforme scroll
- Zoom Focus → Elemento que amplia ao receber foco
- Isolated Element → Elemento sozinho com muito espaco negativo
- Framed Content → Conteudo enquadrado por elementos decorativos
- Contained Center → Container central com margens generosas

### Baseados em Movimento
- Kinetic → Elementos em movimento constante
- Reactive → Elementos que reagem a acoes do usuario
- Ambient Motion → Movimento sutil de fundo sempre presente
- Rhythmic → Movimento com ritmo/padrao definido
- Organic Flow → Movimento que imita natureza (agua, vento)
- Mechanical → Movimento preciso, como maquina
- Chaotic → Movimento aleatorio controlado
- Breathing → Elementos que "respiram" (expand/contract)

### Baseados em Tipografia
- Type Hero → Tipografia como elemento principal do hero
- Editorial → Layout de revista com colunas e tipografia elaborada
- Poster → Layout de cartaz, tipografia dramatica
- Kinetic Type → Tipografia animada como protagonista
- Variable Type → Fonte variavel que muda com interacao
- Type as Image → Texto tratado como elemento visual/arte
- Mixed Type → Multiplas fontes em contraste
- Monospace Editorial → Tipografia monoespacada como design

### Baseados em Midia
- Photo Essay → Sequencia de fotos conta historia
- Video Immersive → Video como fundo ou protagonista
- Gallery Wall → Multiplas imagens como galeria de arte
- Lookbook → Estilo de catalogo de moda
- Documentary → Midia com legendas/contexto jornalistico
- Collage → Multiplas midias sobrepostas
- Before/After → Comparacao visual de estados
- Case Study → Midia mostrando processo/resultado

### Baseados em Interacao
- Cursor Playground → Cursor e parte da experiencia
- Drag Interface → Elementos arrastaveis
- Reveal on Demand → Conteudo aparece com interacao
- Gamified → Elementos de jogo na interface
- Conversational → Interface como dialogo
- Explorable → Usuario explora livremente
- Quiz/Assessment → Interacao que gera resultado
- Configurator → Usuario customiza algo visualmente

### Baseados em Densidade
- Minimal → Pouquissimos elementos, maximo respiro
- Sparse → Elementos bem espacados
- Balanced → Equilibrio entre cheio e vazio
- Dense → Muita informacao organizada
- Maximalist → Tudo, camadas, texturas, cores
- Data Dense → Dashboard, muitos dados visiveis
- White Space Hero → Espaco vazio como protagonista
- Horror Vacui → Medo do vazio, tudo preenchido

## REGRAS DE USO

1. **Para CADA secao**, escolha pelo menos:
   - 1 Arquetipo de composicao
   - 2 Constraints criativos de categorias diferentes

2. **NUNCA repita** o mesmo arquetipo em secoes consecutivas

3. **DECLARE** suas escolhas antes de implementar:
   "Secao X: Arquetipo [Y] + Constraints [A, B]"

4. **Para FONTES**, escolha da lista de pairings curados por contexto de uso:
   - NUNCA use fontes overused: Fraunces, Playfair Display, Montserrat, Poppins, Roboto, Lato, Raleway, Lora, Inter, Open Sans, Merriweather, Source Sans Pro, Oswald
   - NUNCA use combinacoes genericas: Montserrat + Open Sans, Poppins + Roboto, Lato + Lato
   - Escolha a CATEGORIA que melhor se encaixa no projeto (Editorial, Brutalist, Agnostic, Avant-Garde, Cinematic)
   - Use os PESOS especificos indicados nas combinacoes
   - DECLARE: "Fontes: [Heading] + [Body] - [Categoria]"

5. **EVITE** os padroes mais comuns:
   - 3 cards lado a lado
   - Grid simetrico com icones
   - Hero centralizado generico
   - Secao de features com bullets
