# Animated List

List items animate in with stagger effect on scroll.

## Usage

Add `.al-list` with `.al-item` children. Items reveal with stagger as they scroll into view.

## CSS Custom Properties

| Property | Default | Description |
|-|-|
| `--al-stagger` | `80ms` | Stagger delay between items |
| `--al-distance` | `20px` | Slide-up distance |
