Tabs

Animated tabs with a sliding indicator and smooth content transitions.

Product Overview

A comprehensive effects library built with vanilla JS and CSS. Zero dependencies, maximum performance. Every effect is production-ready with full accessibility support and responsive design built in from the ground up.

Key Features

CSS custom properties for theming, IntersectionObserver for performance, prefers-reduced-motion support, mobile-first responsive design, and a clean API for runtime configuration. Each effect works independently.

Easy Integration

Copy the HTML, CSS, and JS files into your project. No build step required. Works with any framework or static site generator. Simply include the files and add the required markup to your page.
