# Glass Card

Frosted glass card with backdrop blur, animated gradient border glow on hover, and subtle elevation shift. Pure CSS effect.

## Usage

Add `.glass-card` class. Works best over a background with color blobs or gradients to showcase the blur effect.

## CSS Custom Properties

| Property | Default | Description |
|-|-|
| `--glass-blur` | `16px` | Blur intensity |
| `--glass-bg` | `rgba(255,255,255,0.05)` | Glass tint |
| `--glass-border` | `rgba(255,255,255,0.1)` | Border color |
| `--glass-glow` | `rgba(139,92,246,0.4)` | Hover glow |
| `--glass-radius` | `20px` | Border radius |
