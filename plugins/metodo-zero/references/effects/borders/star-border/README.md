# Star Border

Canvas-based sparkle particles floating along element perimeter. Each star has independent speed, size, and flicker timing.

## Usage

Add `.star-bdr` with `<canvas class="star-bdr__canvas"></canvas>` inside. JS calculates perimeter path including rounded corners.

## CSS Custom Properties

| Property | Default | Description |
|-|-|
| `--star-bdr-color` | `#fbbf24` | Star/sparkle color |
| `--star-bdr-size` | `4px` | Base star size |
| `--star-bdr-count` | `15` | Number of stars |
| `--star-bdr-speed` | `8s` | Movement speed |
| `--star-bdr-radius` | `16px` | Border radius |
