Accordion

Smooth animated accordion with auto-height transition and rotating chevrons.

This accordion uses real auto-height measurement instead of fixed max-heights. The content area smoothly transitions to the exact height of its content, creating a natural, fluid animation regardless of content length.
Yes. Uses semantic button triggers, proper ARIA attributes can be added easily, and the component respects prefers-reduced-motion by disabling transitions when the user prefers reduced motion.
The accordion measures content height on each open action, so it adapts to dynamically loaded content, images, or any content that changes size. No hardcoded heights needed.
By default, opening one item closes the others (single-open mode). This can easily be modified to allow multiple open items by removing the close-all logic in the click handler.
