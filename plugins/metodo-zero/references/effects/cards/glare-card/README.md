# Glare Card

Glossy glare overlay follows cursor with subtle 3D tilt. Simulates a polished glass surface reflecting light.

## Usage

Wrap `.glare-card` in `.glare-wrapper` (perspective context). Add `.glare-card__glare` div for the light overlay.

## CSS Custom Properties

| Property | Default | Description |
|-|-|
| `--glare-max-opacity` | `0.3` | Maximum glare intensity |
| `--glare-size` | `250px` | Glare spotlight size |
| `--glare-bg` | `#13132b` | Card background |
| `--glare-radius` | `16px` | Border radius |
| `--glare-tilt` | `8deg` | 3D tilt amount |
