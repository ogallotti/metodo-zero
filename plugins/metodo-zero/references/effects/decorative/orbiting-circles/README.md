# Orbiting Circles

Circles orbiting around a center point at different radii and speeds. Pure CSS animation.

## Usage

Add `.oc-stage` with `.oc-orbit` ring elements and `.oc-dot` orbiting dots. All animation is CSS-driven.

## CSS Custom Properties

| Property | Default | Description |
|-|-|
| `--oc-color-1` | `#6366f1` | Inner orbit color |
| `--oc-color-2` | `#ec4899` | Middle orbit color |
| `--oc-color-3` | `#06b6d4` | Outer orbit color |
| `--oc-radius-1` | `100px` | Inner orbit radius |
| `--oc-radius-2` | `160px` | Middle orbit radius |
| `--oc-radius-3` | `220px` | Outer orbit radius |
| `--oc-speed` | `8s` | Base animation speed |
