# Animated Border Button

Rotating conic-gradient border with hover glow. Uses `@property` for smooth angle animation with fallback rotation.

## Usage

Add `.abtn` button with `.abtn__inner` span. Override colors via inline custom properties.

## CSS Custom Properties

| Property | Default | Description |
|-|-|
| `--abtn-color-1` | `#6366f1` | Gradient color 1 |
| `--abtn-color-2` | `#ec4899` | Gradient color 2 |
| `--abtn-speed` | `3s` | Rotation speed |
| `--abtn-border` | `2px` | Border width |
| `--abtn-radius` | `12px` | Border radius |
| `--abtn-bg` | `#0f0f1a` | Inner background |
