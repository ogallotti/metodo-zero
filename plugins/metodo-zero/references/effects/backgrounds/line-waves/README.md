# Line Waves

Horizontal sine wave lines stacked vertically across the screen. Each line has a unique phase, frequency, and amplitude, creating a mesmerizing flowing pattern.

## When to use

- Music or audio-related hero sections
- Calm, meditative landing pages
- Sections needing flowing, elegant motion

## Integration

```html
<div class="lwv-container">
  <canvas class="lwv-canvas"></canvas>
  <div class="lwv-content">
    <!-- Your content here -->
  </div>
</div>
```

## Notes

- Multi-frequency sine composition per line for natural-looking waves
- Horizontal gradient fade on each line prevents hard edges
- Mouse proximity creates ripple distortion on nearby lines
- Center lines are brighter, edges fade for depth
