Floating Dock

Fixed bottom bar with smooth hover interactions. Scroll to see it in action.

The floating dock remains anchored at the bottom of the viewport as you scroll. Each icon responds to hover with a subtle lift animation and background highlight. Click any icon to mark it as active.

This pattern is commonly used for primary navigation in single-page applications, providing persistent access to core actions without consuming header space.

The dock uses backdrop-filter blur for a glassmorphism effect that lets the content behind it show through with a frosted glass appearance, maintaining visual context while staying prominent.
