# Sparkles

Twinkling sparkle particles that appear and fade across a container.

## Usage

Add class `sp-container` to any element. Sparkles will spawn within its bounds.

## CSS Custom Properties

| Property | Default | Description |
|-|-|
| `--sp-color` | `#fbbf24` | Sparkle color |
| `--sp-size-min` | `4px` | Minimum sparkle size |
| `--sp-size-max` | `12px` | Maximum sparkle size |

## JS Parameters

| Param | Default | Description |
|-|-|
| `density` | `20` | Number of sparkles |
| `speed` | `1.5` | Twinkle duration in seconds |
