# Reflective Card

Card with holographic-style reflective sheen that follows cursor movement. Rainbow overlay shifts angle based on cursor position.

## Usage

Add `.reflective-card` with `.reflective-card__sheen` and `.reflective-card__rainbow` overlay divs inside.

## CSS Custom Properties

| Property | Default | Description |
|-|-|
| `--ref-sheen-size` | `300px` | Sheen radius |
| `--ref-sheen-color-1` | `rgba(255,255,255,0.15)` | Primary sheen |
| `--ref-sheen-color-2` | `rgba(120,119,198,0.1)` | Secondary sheen |
| `--ref-bg` | `#141428` | Card background |
| `--ref-radius` | `16px` | Border radius |
