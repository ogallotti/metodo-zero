# Magic Bento

Bento grid cards with spotlight hover effect that illuminates card borders on cursor proximity.

## Usage

Add `.mb-grid` with `.mb-card` children. Script tracks mouse position across the entire grid.

## CSS Custom Properties

| Property | Default | Description |
|-|-|
| `--mb-gap` | `16px` | Grid gap |
| `--mb-radius` | `16px` | Card radius |
| `--mb-spotlight-color` | `#6366f1` | Border glow color |
| `--mb-spotlight-size` | `300px` | Spotlight radius |
