# Following Pointer

Element that smoothly follows cursor with spring physics for natural bouncy motion.

## Usage

Add a `<div class="fp-follower"></div>` element. Script handles physics simulation and trail dots.

## CSS Custom Properties

| Property | Default | Description |
|-|-|
| `--fp-size` | `40px` | Follower diameter |
| `--fp-color` | `#6366f1` | Follower color |
| `--fp-border-color` | `#818cf8` | Border color |

## JS Parameters

| Param | Default | Description |
|-|-|
| `stiffness` | `0.15` | Spring stiffness |
| `damping` | `0.7` | Spring damping |
