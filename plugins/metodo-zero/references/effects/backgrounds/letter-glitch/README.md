# Letter Glitch

Screen filled with randomly changing ASCII characters overlaid with bursts of colored glitch blocks and occasional horizontal screen tears.

## When to use

- Cyberpunk or hacker-themed hero sections
- Game or entertainment landing pages
- Developer conference or hackathon sites

## Integration

```html
<div class="lgl-container">
  <canvas class="lgl-canvas"></canvas>
  <div class="lgl-vignette"></div>
  <div class="lgl-content">
    <!-- Your content here -->
  </div>
</div>
```

## Notes

- Characters change at random intervals for living-code feel
- Glitch blocks spawn in bursts with random size, color, and lifetime
- Some glitch blocks include colored text overlays
- CSS vignette overlay adds depth and focus to center content
