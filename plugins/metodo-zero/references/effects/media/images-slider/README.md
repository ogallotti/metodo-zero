# Images Slider

Horizontal image slider with smooth transitions, navigation arrows, and dot indicators.

## Usage

Add `.is-slider` with `.is-track` containing `.is-slide` elements. Arrows and dots auto-initialize.

## CSS Custom Properties

| Property | Default | Description |
|-|-|
| `--is-radius` | `16px` | Slider border radius |
| `--is-accent` | `#6366f1` | Active dot color |

## JS Parameters

| Param | Default | Description |
|-|-|
| `autoplay` | `4000` | Autoplay interval (0 = off) |

## Features

- Touch swipe support
- Keyboard navigation via arrow keys
- Autoplay pause on offscreen
- Data-caption attributes for slide captions
