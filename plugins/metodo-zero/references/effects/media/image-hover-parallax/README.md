# Image Hover Parallax

Image shifts with subtle 3D parallax tilt on hover, responding to cursor position.

## Usage

Add `.ihp-card` elements. Script auto-applies tilt on hover.

## CSS Custom Properties

| Property | Default | Description |
|-|-|
| `--ihp-radius` | `16px` | Card border radius |
| `--ihp-shadow` | `30px` | Shadow depth on hover |

## JS Parameters

| Param | Default | Description |
|-|-|
| `intensity` | `15` | Max tilt angle in degrees |
| `scale` | `1.05` | Scale factor on hover |
