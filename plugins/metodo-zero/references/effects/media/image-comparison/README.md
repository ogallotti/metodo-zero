# Image Comparison

Drag slider to compare two images side by side with a smooth divider line.

## Usage

Add `.ic-wrapper` with two images (`.ic-img--before` and `.ic-img--after`), a `.ic-divider`, and `.ic-handle`.

## CSS Custom Properties

| Property | Default | Description |
|-|-|
| `--ic-divider-color` | `#ffffff` | Divider line color |
| `--ic-divider-width` | `3px` | Divider line width |
| `--ic-handle-size` | `44px` | Drag handle diameter |
| `--ic-radius` | `16px` | Container border radius |
