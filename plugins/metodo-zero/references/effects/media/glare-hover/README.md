# Glare Hover

Image or card with glossy reflective glare effect that moves with cursor on hover.

## Usage

Add `.gh-card` with a `.gh-glare` overlay div inside. Script tracks mouse position.

## CSS Custom Properties

| Property | Default | Description |
|-|-|
| `--gh-glare-color` | `#ffffff` | Glare color |
| `--gh-glare-opacity` | `0.25` | Glare intensity |
| `--gh-glare-size` | `60%` | Glare radius |
| `--gh-radius` | `16px` | Card border radius |
