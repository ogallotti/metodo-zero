# Border Trail

Glowing trail with fading tail following border path. Uses `conic-gradient` with `@property` angle animation and blurred outer glow.

## Usage

Add `.trail-box` class. Trail length and color are configurable via custom properties.

## CSS Custom Properties

| Property | Default | Description |
|-|-|
| `--trail-color` | `#a855f7` | Trail color |
| `--trail-length` | `90deg` | Trail arc length |
| `--trail-speed` | `4s` | Animation speed |
| `--trail-width` | `2px` | Border width |
| `--trail-radius` | `16px` | Border radius |
| `--trail-bg` | `#111125` | Background |
