# Counter

Animated counter display with eased number transitions.

## CSS Custom Properties

| Property | Default | Description |
|-|-|
| `--cnt-font-size` | `4rem` | Number font size |
| `--cnt-color` | `#fafafa` | Number color |

## JS Parameters

| Param | Default | Description |
|-|-|
| `duration` | `2000` | Animation duration in ms |
