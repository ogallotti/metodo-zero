Cards Cascade

Cards waterfall into view from alternating sides with stagger, creating a dynamic and energetic scroll experience.

Dynamic Energy

The alternating slide direction and staggered timing create a sense of movement and energy. Each card slightly overlaps the previous one's timing, building momentum as the user scrolls through the content.

Lightweight and Performant

IntersectionObserver triggers each card's entrance. CSS transitions handle the animation with GPU-accelerated transforms. No scroll listeners, no layout thrashing, no heavy libraries.

Pure vanilla JS with IntersectionObserver and CSS transitions.
