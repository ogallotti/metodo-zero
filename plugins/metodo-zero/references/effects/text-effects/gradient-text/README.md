# Gradient Text

Animated gradient flowing through text. Pure CSS effect with JS only for IntersectionObserver pause.

## Usage

```html
<h1 class="gt-text">Your headline here.</h1>
```

## CSS Custom Properties

| Property | Default | Description |
|-|-|-|
| `--gt-font-size` | `4.5rem` | Font size |
| `--gt-color-1` | `#6ee7b7` | First gradient color |
| `--gt-color-2` | `#3b82f6` | Second gradient color |
| `--gt-color-3` | `#a78bfa` | Third gradient color |
| `--gt-color-4` | `#f472b6` | Fourth gradient color |
| `--gt-speed` | `4s` | Animation speed |
| `--gt-angle` | `135deg` | Gradient angle |
| `--gt-bg` | `#0a0a0a` | Background color |

## Features

- Pure CSS animated gradient
- Four customizable colors
- Adjustable angle and speed
- IntersectionObserver pauses when offscreen
- prefers-reduced-motion support
- Mobile responsive
