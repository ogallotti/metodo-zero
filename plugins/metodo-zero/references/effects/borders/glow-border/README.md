# Glow Border

Soft pulsing glow border using `box-shadow` animation. Glow locks at peak intensity on hover.

## Usage

Add `.glow-bdr` class. Variants: `--purple`, `--emerald`.

## CSS Custom Properties

| Property | Default | Description |
|-|-|
| `--glow-bdr-color` | `#6366f1` | Glow color |
| `--glow-bdr-intensity` | `0.4` | Glow opacity |
| `--glow-bdr-blur` | `12px` | Blur radius |
| `--glow-bdr-speed` | `2.5s` | Pulse speed |
| `--glow-bdr-width` | `1px` | Border width |
| `--glow-bdr-radius` | `16px` | Border radius |
