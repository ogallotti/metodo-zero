# Rainbow Button

Continuously animated rainbow gradient border with glow. Inner background disappears on hover to reveal full gradient. Pure CSS.

## Usage

Add `.rainbow-btn` with `.rainbow-btn__inner` span. Variant: `--filled` for always-visible gradient fill.

## CSS Custom Properties

| Property | Default | Description |
|-|-|
| `--rain-speed` | `3s` | Animation speed |
| `--rain-border-width` | `2px` | Gradient border width |
| `--rain-radius` | `12px` | Border radius |
| `--rain-bg` | `#0f0f1a` | Inner background |
| `--rain-glow-blur` | `12px` | Glow blur |
