Parallax Hero

Multiple layers at different scroll speeds create a stunning depth illusion

Layers Create Depth

The parallax effect distributes visual elements across five independent layers. Each layer moves at a different rate relative to the user's scroll, tricking the brain into perceiving three-dimensional space on a flat screen. The closest layers move fastest while the background barely shifts.

Performance First

Only translateY transforms are applied per frame, keeping all animations on the GPU compositor layer. Passive scroll listeners and requestAnimationFrame throttle ensure smooth 60fps even with many layers active simultaneously.

Customizable Speeds

Each layer's speed is driven by a CSS custom property, letting designers tune the depth relationship without touching JavaScript. Negative values invert the direction for creative effects like elements that float upward as you scroll down.

A Canvas for Creativity

Swap the geometric shapes for images, illustrations, or SVGs to match your brand. The layering system accepts any content and applies the same depth mechanics regardless of what is inside each layer.
