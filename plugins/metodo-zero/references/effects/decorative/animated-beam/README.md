# Animated Beam

SVG beam connecting two points with animated dash pattern and glow effect.

## Usage

Add `.ab-scene` with `.ab-svg` and `.ab-node` elements. Beams are auto-drawn between nodes and the center.

## CSS Custom Properties

| Property | Default | Description |
|-|-|
| `--ab-color` | `#6366f1` | Beam color |
| `--ab-width` | `2px` | Beam stroke width |
| `--ab-glow` | `8px` | Glow blur size |
| `--ab-speed` | `2s` | Dash animation duration |
