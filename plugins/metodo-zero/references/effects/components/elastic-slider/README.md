# Elastic Slider

Slider with elastic/bouncy snap behavior and animated fill.

## CSS Custom Properties

| Property | Default | Description |
|-|-|
| `--es-accent` | `#6366f1` | Fill and thumb accent |
| `--es-track-height` | `6px` | Track height |

## JS Parameters

| Param | Default | Description |
|-|-|
| `snap` | `10` | Number of snap points |
| `elasticity` | `0.3` | Bounce factor |
