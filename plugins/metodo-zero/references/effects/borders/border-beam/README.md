# Border Beam

Animated light beam traveling along border using `conic-gradient` with `@property` for smooth angle animation. Fallback uses rotating pseudo-element.

## Usage

Add `.beam-box` class. Pure CSS with `mask-composite` for border-only rendering.

## CSS Custom Properties

| Property | Default | Description |
|-|-|
| `--beam-color` | `#6366f1` | Beam color |
| `--beam-size` | `80px` | Beam length |
| `--beam-speed` | `6s` | Animation speed |
| `--beam-border-width` | `2px` | Border width |
| `--beam-radius` | `12px` | Border radius |
| `--beam-bg` | `#111122` | Background |
