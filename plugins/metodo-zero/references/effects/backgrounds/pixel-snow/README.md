# Pixel Snow

Falling pixel square snow particles with horizontal wind drift and accumulation at the bottom edge. Calm, cozy aesthetic.

## When to use

- Winter or holiday themed hero sections
- Cozy, calm landing page backgrounds
- Any section wanting a gentle, atmospheric touch

## Integration

```html
<div class="psn-container">
  <canvas class="psn-canvas"></canvas>
  <div class="psn-content">
    <!-- Your content here -->
  </div>
</div>
```

## Notes

- Snow accumulates at the bottom with neighbor spreading for smooth piles
- Slow melting prevents endless growth of accumulated snow
- Wind oscillates sinusoidally for natural gusting effect
- Each flake has unique wobble frequency and drift
