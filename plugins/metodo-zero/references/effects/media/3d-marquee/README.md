# 3D Marquee

3D perspective marquee of images/cards that scroll continuously with depth and rotation effects.

## Usage

Add `.m3d-scene` with `.m3d-track-wrapper` containing `.m3d-track` rows of `.m3d-item` cards. Script auto-duplicates items for seamless looping.

## CSS Custom Properties

| Property | Default | Description |
|-|-|
| `--m3d-speed` | `30s` | Scroll animation duration |
| `--m3d-gap` | `24px` | Gap between items |
| `--m3d-perspective` | `800px` | 3D perspective depth |
| `--m3d-rotate` | `5deg` | X-axis rotation angle |
