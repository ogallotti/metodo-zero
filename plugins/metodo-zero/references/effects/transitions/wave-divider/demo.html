Wave Divider

Animated SVG waves create fluid, organic transitions between sections. The continuous motion adds life to static page layouts.

Flowing motion

The wave slides horizontally in an infinite loop. Two identical wave patterns are tiled side by side, so the translateX animation creates a seamless loop.

Adjust speed with --wave-speed, height with --wave-height.

Layered waves

Reverse the animation direction for variety. Alternating wave directions between sections creates visual rhythm and prevents monotony.

Color harmony

Wave fills automatically match the next section's background color. The transition feels natural because the wave literally flows into the next section.

Endless possibilities

Customize wave shapes by editing the SVG path data. Gentle curves, sharp peaks, or asymmetric flows — the wave divider adapts to any visual language.
