# Shiny Button

Metallic shine sweep on hover. Single-pass animation triggered by `:hover`. Pure CSS.

## Usage

Add `.shiny-btn` class. Variants: `--dark`, `--gradient`.

## CSS Custom Properties

| Property | Default | Description |
|-|-|
| `--shiny-bg` | `#6366f1` | Button background |
| `--shiny-color` | `#ffffff` | Text color |
| `--shiny-shine-color` | `rgba(255,255,255,0.3)` | Shine streak color |
| `--shiny-speed` | `500ms` | Sweep duration |
| `--shiny-radius` | `12px` | Border radius |
