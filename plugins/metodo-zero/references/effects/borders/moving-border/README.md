# Moving Border

Solid border segment that orbits the perimeter continuously. Uses `conic-gradient` with `@property` angle animation and mask-composite for border-only rendering.

## Usage

Add `.mov-bdr` class. Pure CSS effect.

## CSS Custom Properties

| Property | Default | Description |
|-|-|
| `--mov-color` | `#6366f1` | Segment color |
| `--mov-segment` | `60deg` | Segment arc size |
| `--mov-speed` | `5s` | Orbit speed |
| `--mov-width` | `2px` | Border width |
| `--mov-radius` | `12px` | Border radius |
| `--mov-bg` | `#111122` | Background |
