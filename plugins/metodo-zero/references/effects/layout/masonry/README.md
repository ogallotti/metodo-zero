# Masonry

Animated masonry grid layout with scroll-triggered reveal.

## Usage

Add `.ms-grid` with `.ms-item` children. CSS column-count handles the masonry layout.

## CSS Custom Properties

| Property | Default | Description |
|-|-|
| `--ms-columns` | `3` | Number of columns |
| `--ms-gap` | `16px` | Grid gap |
| `--ms-radius` | `12px` | Card border radius |
