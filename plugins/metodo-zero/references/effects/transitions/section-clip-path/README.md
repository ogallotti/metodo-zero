# Section Clip Path

Section transitions with animated clip-path shapes.

## Dependencies

None.

## Variants

- `.scp-clip--arrow`: Downward-pointing chevron
- `.scp-clip--scoop`: Elliptical curve
- `.scp-clip--slash`: Diagonal cut
- `.scp-clip--diamond`: Geometric diamond notch

## CSS Custom Properties

| Property | Default | Description |
|-|-|-|
| `--scp-duration` | `600ms` | Content fade-in duration |
| `--scp-overlap` | `80px` | How much sections overlap |

## Accessibility

- Clip-paths removed with `prefers-reduced-motion`
