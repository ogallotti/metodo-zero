# Glow Button

Pulsing/breathing glow that intensifies on hover. Uses blurred pseudo-element with scale animation. Pure CSS.

## Usage

Add `.glow-btn` class. Color variants: `--green`, `--pink`, `--amber`.

## CSS Custom Properties

| Property | Default | Description |
|-|-|
| `--glow-btn-bg` | `#6366f1` | Button background |
| `--glow-btn-color` | `#ffffff` | Text color |
| `--glow-btn-glow-color` | `rgba(99,102,241,0.5)` | Glow color |
| `--glow-btn-speed` | `2s` | Pulse speed |
| `--glow-btn-blur` | `20px` | Glow blur radius |
| `--glow-btn-radius` | `12px` | Border radius |
