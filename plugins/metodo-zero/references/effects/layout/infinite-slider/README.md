# Infinite Slider

Infinite horizontal scrolling content with smooth seamless looping.

## Usage

Add `.inf-container` with `.inf-track` containing `.inf-item` elements. Items are auto-duplicated for seamless loop.

## CSS Custom Properties

| Property | Default | Description |
|-|-|
| `--inf-speed` | `25s` | Scroll animation duration |
| `--inf-gap` | `32px` | Item gap |
| `--inf-direction` | `normal` | Scroll direction |
