# Hero Video Dialog

Video thumbnail that expands into a modal dialog with smooth animation.

## Usage

Add `.hvd-thumbnail` for the clickable preview and `.hvd-modal` for the dialog. Script handles open/close.

## CSS Custom Properties

| Property | Default | Description |
|-|-|
| `--hvd-radius` | `16px` | Thumbnail border radius |
| `--hvd-overlay` | `rgba(0,0,0,0.85)` | Backdrop color |
| `--hvd-accent` | `#6366f1` | Play button color |
