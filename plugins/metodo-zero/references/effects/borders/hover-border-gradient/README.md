# Hover Border Gradient

Border transitions from subtle monochrome to vivid multi-color gradient on hover. Uses `background-position` shift for smooth gradient animation. Pure CSS.

## Usage

Add `.hbg-box` with `.hbg-box__glow` div for outer glow. Override gradient colors via inline custom properties.

## CSS Custom Properties

| Property | Default | Description |
|-|-|
| `--hbg-color-1` | `#6366f1` | Gradient stop 1 |
| `--hbg-color-2` | `#ec4899` | Gradient stop 2 |
| `--hbg-color-3` | `#f59e0b` | Gradient stop 3 |
| `--hbg-speed` | `400ms` | Transition speed |
| `--hbg-width` | `2px` | Border width |
| `--hbg-radius` | `12px` | Border radius |
| `--hbg-bg` | `#111122` | Background |
