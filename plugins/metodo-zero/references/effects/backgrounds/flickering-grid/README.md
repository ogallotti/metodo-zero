# Flickering Grid

Grid of cells with staggered flickering opacity animation, dynamic sizing, and a radial vignette fade.

## When to use

- Tech/data dashboard hero sections
- Backgrounds with a matrix/pixel aesthetic
- Sections needing a subtle, structured motion

## Integration

```html
<div class="fg-container">
  <div class="fg-grid"></div>
  <div class="fg-vignette"></div>
  <div class="fg-content">
    <!-- Your content here -->
  </div>
</div>
```

## Notes

- Grid cells are dynamically generated by JS to fill the viewport
- 20 staggered animation delays create organic, non-uniform flickering
- Vignette layer fades the grid edges into the background
- Cell size and gap are CSS custom properties for easy customization
- Uses CSS Grid `auto-fill` for responsive layout
